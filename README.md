## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Clone the project using git@github.com:desireemendes/tinyurl.git
- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
